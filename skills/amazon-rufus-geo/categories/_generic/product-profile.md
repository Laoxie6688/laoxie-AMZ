# 通用 Product Profile

每个 ASIN 至少记录：

- ASIN / role / marketplace / URL
- title / brand / price / rating / review_count / BSR
- bullets
- Product Overview / detail bullets
- A+ 摘要和模块数量
- 主图数量、场景图、参数图、对比图是否存在
- 当前 Q&A 摘要
- review themes：好评主题、差评主题、尺寸/质量/使用相关投诉
- included_components、material、dimensions、unit_count
- not_captured 字段：抓不到就写 `not_captured`，不要猜
