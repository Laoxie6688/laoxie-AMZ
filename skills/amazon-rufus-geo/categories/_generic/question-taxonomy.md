# 通用兜底 Question Taxonomy

用于未建立专属类目包的实物商品。稳定类目应新建专属 pack。

| 维度 | 典型问题 | Listing 证据 |
|---|---|---|
| specs | What are the dimensions? What's included? What's it made of? | 标题、Product Overview、参数图、A+ specs |
| usage | What is it used for? Is it easy to set up/use? | 场景图、五点、A+ how-to |
| quality | Is it sturdy? How long does it last? | 材质、工艺、保修、评论 |
| complaint | What are common problems? Does it match photos? | 差评、Q&A、FAQ 图 |
| vs_competitor | What makes it different? Which one is better for X? | A+ 对比表、五点边界说明 |

优先级：差评和品质类问题优先级最高；其次是规格和竞品对比。
