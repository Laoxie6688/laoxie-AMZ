# Pet Supplies Question Taxonomy

用于 dog harness / pet travel kit / pet training / cat litter box 等宠物用品。

## 维度与典型 Rufus 问法

| 维度 | 典型问法 | Rufus 需要的证据 |
|---|---|---|
| fit_size | What size should I choose? Is it good for large dogs? | 尺寸表、neck/chest girth、体型边界 |
| no_pull_control | Does it help with pulling? Is it good for training? | front/back clip、受力机制、训练场景 |
| comfort_pressure | Does it choke the dog? Is it comfortable for daily walks? | 胸背受力、padding、breathable mesh、材质 |
| travel_safety | Does it include a seat belt? Is it good for car rides/hiking? | seat belt、reflective、travel bowl、road trip 场景 |
| accessories_included | What comes in the set? Is it a complete kit? | 配件清单、哪些是高频必需品 |
| durability | Are the buckles/rings/stitching strong? | 五金、缝线、尼龙、质保或评论证据 |
| cleaning_care | Is it washable? Does it smell? | 清洁说明、材质、评论 |
| complaint | What are common issues? Does it run small? | 差评、Q&A、FAQ 预防 |
| vs_competitor | Which is better for X? How does it compare? | 对比表、适合/不适合边界 |

## Dog Harness 重点问题

必须覆盖：

1. 尺寸怎么选：neck girth、chest girth，不只按品种/体重。
2. 日常 no-pull 与 extreme heavy puller 的边界。
3. 是否有 front/back clip，是否适合 training walks。
4. 是否减少 neck pressure / choking。
5. 是否适合 car rides、road trips、hiking、night walks。
6. 配件是 essential kit 还是 random extras。
7. 和 basic harness、tactical harness、extra-accessory bundle 的推荐分流。
