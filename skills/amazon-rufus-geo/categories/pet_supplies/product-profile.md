# Pet Supplies Product Profile

每个 ASIN 建档时记录：

- ASIN / role / marketplace / URL / parent-child ASIN relationship
- title / brand / price / rating / review_count / BSR
- product type：dog harness、harness set、cat litter box、carrier 等
- target_species、breed_size、life_stage
- size chart：neck girth、chest girth、weight range、size variants
- included_components：harness、collar、leash、seat belt、bowl、dispenser、toy 等
- material：nylon、mesh、metal buckle、padding、hardware
- attachment/control：front clip、back clip、handle、D-ring、triangular ring
- comfort evidence：padding、breathable mesh、pressure distribution、no choke
- travel evidence：seat belt、reflective、portable bowl、car rides、hiking
- durability evidence：buckles、stitching、rust-proof、chew resistance
- cleaning/care evidence：washable、odor、easy clean
- A+ modules：组件图、尺寸图、场景图、对比图、FAQ 是否存在
- Q&A themes：尺寸、控制、出行、耐用、清洁、竞品对比
- review themes：好评场景、差评原因、退货风险
- not_captured：抓不到的字段明确写 `not_captured`
