## 一、两层关键词策略概览

| 层级 | 用途 | 核心逻辑 | 落点 |
|------|------|---------|------|
| **传统关键词（A9/A10 兼容层）** | 传统搜索排名 | 关键词匹配 + 搜索量 | 标题、五点、后台搜索词 |

### Step 3：后台搜索词（250字节内）
**填写规则：**
- 不重复标题中已有的词
- 用空格分隔，不用逗号
- 包含拼写变体（如 colour/color）
- 包含同义词（如 bag/pack/rucksack）
- 包含常见错别字
- 不写品牌词、ASIN、违禁词

## 三、Rufus 语义关键词（GEO 层）

这是传统 SEO 没有的新层。核心原则：**名词短语 > 单词关键词，场景化 > 规格参数，问题型 > 陈述型**。

### 对比示例（以登山包为例）

| ❌ 传统关键词 | ✅ Rufus 语义关键词 |
|---|---|
| waterproof hiking backpack | "backpack for a weekend camping trip in the rain" |
| 60L backpack | "60L pack that holds 4 days of gear without bulging" |
| ultralight backpack | "for thru-hikers who count every ounce" |
| durable backpack | "PCT-tested pack that survives 2,650 miles" |
| camping backpack | "What do I need for camping?" |

### 按 COSMO 关系生成语义关键词

基于 Step 2（15 问模拟）的结果，为每个 ✅ 或 ⚠️ 的关系生成 1-2 个语义关键词短语：

| COSMO 关系 | 语义关键词短语 | 落点 |
|-----------|--------------|------|
| USED_FOR_EVE | | 标题/五点/Q&A |
| USED_FOR_AUD | | 五点/A+ |
| CAPABLE_OF | | 五点/Q&A |
| USED_ON | | 五点/后台 Season |
| USED_IN_LOC | | 五点/A+ |
| XWANT | | 标题/后台搜索词 |
| XIS_A | | A+/Q&A |

## 四、关键词优先级分配

| 位置 | 放什么词 | 字符限制 |
|------|---------|---------|
| 标题前 80 字符 | 核心词 + 最重要的 Rufus 语义场景 | 重点！ |
| 标题 80-200 字符 | 次核心词 + 目标人群 + 规格 | |
| 五点描述每条开头 | Rufus 语义关键词（触发词开头） | |
| 产品描述第一段 | 次要语义关键词 + 场景扩展 | |
| 后台搜索词 | 传统长尾词 + 同义词 + 变体 | 250字节 |
| Q&A 问题句 | XWANT / XIS_A / CAPABLE_OF 类语义词 | |
