# 05-description-aplus.md — 产品描述 / A+ 内容写作模板

> **使用时机**：五点描述写完后，写产品描述或 A+ 内容。
> 产品描述的职责是**扩展覆盖五点没提到的 COSMO 关系**，不是重复五点。

---

## 一、产品描述 vs A+ 的区别

| | 产品描述（文字版） | A+ 内容（图文版） |
|---|---|---|
| **适用场景** | 没有 A+ 资质时 / 辅助 A+ | 品牌注册卖家 |
| **Rufus 读取** | ✅ 直接读文字 | ✅ 读文字 + **读图片 alt-text** |
| **字数限制** | 约 2000 字符 | 按模块，无硬性限制 |
| **重点** | 场景扩展 + 信任背书 | 视觉信号 + alt-text 完整性 |

---

## 二、产品描述写作结构（5段式）

### 段落 1：品牌故事 / 产品起源（USED_BY + XINTERESTED_IN）
**目标**：建立信任，打动理念型用户群

写法：
- 为什么做这个产品？（解决了什么真实痛点）
- 品牌的背景或使用者背书
- 不要吹嘘，要叙事

示例：
```
XYZ Pack was designed by a 2,000-mile thru-hiker who couldn't find
a pack that balanced load-carrying comfort with ultralight ethics.
After three prototype iterations on the PCT, the result is a 60L
frameless design that carries heavy loads without a heavy frame.
```

---

### 段落 2：更多使用场景（USED_FOR_EVE + USED_IN_LOC + USED_ON）
**目标**：覆盖五点里没提到的场景，扩大 Rufus 召回范围

写法：
- 列出 3-5 个具体使用场景（越具体越好）
- 每个场景用 1-2 句话描述（不要只列关键词）

示例：
```
Whether you're planning a spring weekend on the AT, a summer summit
attempt in the Cascades, or a winter hut-to-hut traverse in the Alps,
this pack adapts. The removable lid converts to a 5L summit pack for
fast-and-light alpine starts, and the reinforced base holds up to
granite scrambles and muddy creek crossings alike.
```

---

### 段落 3：常见顾虑预先回答（XWANT + 差评挖掘产物）
**目标**：把评论里的高频疑虑，提前用中性诚实的语言回答

写法：
- 从差评/竞品差评里找高频顾虑
- 每个顾虑用 1-2 句话正面回应
- 诚实 > 营销腔（Rufus 能分辨）

示例：
```
A common question: "Is this pack right for shorter torsos?"
The adjustable suspension fits torso lengths from 15 to 21 inches,
with clear fit guides included. Users under 5'4" typically fit the
small frame; those 5'4" and above fit the medium. If you're between
sizes, size down — a snug hipbelt transfers weight better than a
loose one.
```

---

### 段落 4：搭配建议 + 使用技巧（USED_WITH + CAPABLE_OF）
**目标**：覆盖 USED_WITH 关系，把产品放入更大的使用生态

写法：
- 推荐兼容的知名品牌配件（直接写品牌名，Rufus 会识别）
- 给出 1-2 条实用的使用技巧

示例：
```
Pairs best with a 3L Osprey Hydraulics or CamelBak Crux reservoir —
both fit the padded hydration sleeve without stretching. For alpine
routes, the external crampon pouch accommodates Black Diamond Serac
crampons and most 65cm ice axes. Pro tip: pack your rain jacket in
the front shove-it pocket for fast access during afternoon storms.
```

---

### 段落 5：保障 + 行动号召（XWANT 风险消除）
**目标**：降低购买顾虑，完成转化

写法：
- 保修内容（具体，不要写"我们保证质量"这种废话）
- 明确的退换政策
- 简短有力的收尾

示例：
```
Every pack ships with a lifetime repair guarantee and a 60-day
fit guarantee — if the pack doesn't fit after a real trail test,
return it for a full refund. No questions, no return shipping labels.
Built to last, backed to prove it.
```

## 三、A+ 内容模块结构

### A+ 必须包含的模块（按顺序）

| 模块 | 内容 | 关键要求 |
|------|------|---------|
| **第1屏：英雄图** | 产品主图 + 核心卖点标题 | Alt-text 描述场景，不只是产品名 |
| **第2屏：使用场景** | 生活场景图 × 2-3 | 每张 alt-text 完整描述"图片里发生了什么" |
| **第3屏：人群分区** | 2-3 种用户类型 | "For the [用户类型] who ___" |
| **第4屏：功能对比** | 核心卖点图解 | 规格翻译成能力故事（不要参数表） |
| **第5屏：品牌故事** | 品牌背景 + 信任背书 | USED_BY 社会证明 |
| **第6屏：搭配推荐** | "Complete Your Setup" | USED_WITH 关系，推荐兼容品 |

---

## 四、A+ Alt-text 写作规范

⚠️ **这是 Rufus 独立读取的信号源**，绝大多数卖家完全忽略。

**错误写法（描述图片是什么）：**
```
alt="hiking backpack product image"
alt="60L backpack blue color"
```

**正确写法（描述图片里发生了什么）：**
```
alt="A hiker wearing the XYZ 60L pack crossing a suspension bridge
     in lush green rainforest, pack fully loaded for a 5-day trek"

alt="Close-up of the pack's aluminum frame transfer pad against
     a hiker's lower back, showing the weight distribution system
     in use during a steep uphill climb"
```

**Alt-text 写作公式：**
```
[谁] + [在哪] + [做什么] + [产品的作用/状态]
```

## 五、产品描述/A+ 整体自检

- [ ] 产品描述扩展了五点里没有的 COSMO 关系（不是重复）
- [ ] 至少覆盖了 USED_FOR_EVE（更多场景）+ USED_WITH（搭配）+ USED_BY（社会证明）
- [ ] 段落 3 回答了差评里的高频顾虑（从评论挖掘步骤获取）
- [ ] A+ 有至少 2 张**生活场景图**（不只有白底图）
- [ ] **所有 A+ 图片 alt-text 已完整填写**（描述"发生了什么"，不是"图片是什么"）
- [ ] 没有重复五点描述的内容超过 20%

**完成后，进入 Step 7：读取 06-qa-engineering.md，生成 Q&A。**
